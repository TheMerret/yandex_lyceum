Выигрыш

Ограничение времени 10 секунд
Ограничение памяти 64Mb
Ввод стандартный ввод
Вывод стандартный вывод

В субботу, 20 октября, в половине пятого пополудни пароход вошёл в порт Бомбей – на два дня раньше срока.
Мистер Фогг выплатил капитану обещанное вознаграждение, методично записал в своём путевом блокноте эти два дня в графу выигрышей и сошёл на берег.

Напишите функцию, подводящую итог выигрышей и проигрышей.

Функция prize() принимает два позиционных аргумента – хост и порт сервера и произвольное число именованных параметров: названия пароходов и даты их прибытия в порт назначения.
На сервере по указанному адресу находится словарь с ключами – пароходами и значениями – датами прибытия по расписанию.

Функция возвращает словарь: ключи – пароходы, указанные в аргументах функции, а значения – выигрыш или проигрыш в днях, проигрыш – отрицательное значение. Даты в пределах одного месяца. Если пароход пришел по расписанию, такой ключ создавать не нужно.

Пример 1

Ввод Вывод 
# Пример запуска функции:
server = ["127.0.0.1", 5000]
data = {
    "Carnatic": "25.07",
    "Magnolia": "08.08",
    "Mongolia": "09.08",
    "Beth": "26.07"
}
print(prize(*server, **data))
# Данные на сервере:
{
    "Carnatic": "27.07",
    "Magnolia": "08.08",
    "Mongolia": "01.08",
    "Beth": "25.07",
    "Bay": "19.01"
}

{
    "Carnatic": -2,
    "Mongolia": 8,
    "Beth": 1
}

На всех парах

Ограничение времени 1 секунда
Ограничение памяти 64Mb
Ввод стандартный ввод
Вывод weather.json

Тем временем «Монголия» неслась на всех парах по волнам Красного моря к Адену. Филеас Фогг не обращал внимания на штормовую погоду, а уж как перед самым отплытием из Суэца на пароход торопливо поднялся сыщик Фикс, и вовсе не заметил.

Напишите программу для записи прогноза погоды по пути следования.

Формат ввода

Вводится имя файла csv с информацией о погоде, заголовки файла (разделители ̂):
id, дата, время, погода
id, date, time, weather

Формат вывода

В файл weather.json запишите словарь с ключами – датами и значениями – списками видов погоды в обратном алфавитном порядке.

Пример 1

Ввод Вывод 
# Содержимое stdin:
full_steam.csv
# Содержимое файла full_steam.csv:
id^date^time^weather
1^28/07^6^fog
2^5/10^7^cloudy
3^28/07^7^cloudy
4^28/07^17^windy
5^5/10^7^storm
6^5/10^5^sunny
7^28/07^5^partly cloudy
8^28/07^7^rain
9^5/10^7^heavy rain

# Содержимое файла weather.json:
{
    "28/07": [
        "windy",
        "rain",
        "partly cloudy",
        "fog",
        "cloudy"
    ],
    "5/10": [
        "sunny",
        "storm",
        "heavy rain",
        "cloudy"
    ]
}

Пример 2

Ввод Вывод 
# Содержимое stdin:
storm.csv

Морской слон

Ограничение времени 1 секунда
Ограничение памяти 64Mb
Ввод стандартный ввод
Вывод polar_zoo.json

Зеленое море катило на него медленные валы, а волны покруче разбивались об огромные валуны с плеском.
Внезапно один из валунов испустил громкий рев и поднялся. Это оказался трехметровый морской слон, он был весь в морщинах и складках, а между водянистыми глазками покачивался длинный нос, похожий на слоновий, только сильно изуродованный.

Напишите программу для выбора животных по указанным критериям.

Формат ввода

Через аргументы командной строки передаются:
хост и порт сервера;
произвольное количество видов животных.
Также могут быть переданы:
–min_amount – наименьшее количество животных, по умолчанию 0;
–length – наибольшая длина названия местности обитания, по умолчанию 20.

На сервере находится список словарей с ключами:
kind – вид животных;
name – название;
habitat – место обитания;
amount – количество.

Формат вывода

Получив все данные с сервера, отберите животных по условиям из аргументов командной строки: вид находится среди указанных, количество не меньше значения аргумента, длина названия местности обитания не больше указанной. Запишите в файл polar_zoo.json полученную информацию в виде словаря с ключами:
местность – список списков [название, количество].
Списки в значениях по ключам должны быть отсортированы по возрастанию количества, затем по алфавиту названий.
